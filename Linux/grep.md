# grep

> grep 명령은 파일 내에서 지정한 패턴이나 문자열을
>
> 찾은 후에 그 패턴을 포함하고 있는 모든 행을 출력

```shell
grep [option] [pattern] [파일명]
```

<details>
    <summary>
        option
    </summary>
    <li>-i :대소문자를 구분하지 않고 검색한다.</li>
    <li>-v :패턴과 일치하지 않는 행을 출력한다.</li>
    <li>-c :패턴과 일치하는 행의 개수를 출력한다.</li>
    <li>-w :패턴과 단어 단위로 매칭되어야 출력한다.</li>
</details>
